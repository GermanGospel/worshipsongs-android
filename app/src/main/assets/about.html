About

Description

This app brings you most of the Praise & Worship lyrics used by Tamil churches all over the world. Now, you can worship anytime, anywhere with all the lyrics you need.

Why Worship Songs?

Why develop another app when there are tons of similar apps? The short answer is: There are no similar apps. Worship Songs was developed as we couldn't find an app with latest Praise & Worship songs with both Tamil and Romanised text. So, Worship Songs is the world's first mobile app that offers Romanised text for Tamil lyrics. Many Tamil congregations also sing English songs. Worship Songs provides many popular English songs as well so you need not look for another app. And finally, the songs database evolves rapidly. New songs are added often, so you will never miss your favourite song. Did we mention that Worship Songs is open source? Yes, you can be a part of this project. Isn't that great?

What are the features offered?

Worship Songs offers the following features:

  • A large collection of latest Tamil and English Praise & Worship song lyrics.
  • Romanised text for Tamil songs.
  • Ability to quickly search for a song by typing the first few characters in English. For example, typing "Aara" would help you locate the song "Aarathanai Naayagan Neerae" (ஆராதனை நாயகன் நீரே)
  • Increase/decrease the font size, distinguish Tamil text from Romanised by setting a different colour
  • Keep the screen awake while viewing a song.
  • Automatically update the song database during every startup.

NOTES

While starting Worship Songs for the first time, internet connection is required to download the songs database whose size is roughly 4 MB. Ensure that you are connected to Wi-fi or subscribed to a data plan to avoid incurring charges by your Internet provider.

FAQ

Q. I couldn't find a song that I am looking for. What should I do?

Refine your search terms. For example, instead of "Arathanai", try searching for "Aarathanai". Worship Songs names the song titles that matches closely with their Tamil counterparts.

Q. I still couldn't find the song. I bet it's not there in your database. What now?

You can create a new issue at our Github project. Worship Songs is an open source project. Anyone interested can contribute song lyrics. If you are interested, please let us know.

Q. I want to request a new feature. How do I go about that?

The songs database and the app are maintained in separate Github projects. This is because the songs database is shared by both the Android and iOS version of Worship Songs. Worship Songs for Android lives at https://github.com/crunchersaspire/worshipsongs. You can request for a new feature by creating an issue there.

Q. How do I report bugs?

Please create an issue and apply the label "Bug" in our Github project. Kindly refer the previous question for project details. Please do not report bugs through comments.

Q. How can I help you guys?

If you find Worship Songs to be useful, please give it a good rating and share your opinion. Please help others know about this app by spreading word. You can contribute song lyrics or if you are good at Android/iOS development, you can contribute code.
